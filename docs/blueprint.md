# **App Name**: NEU

## Core Features:

- Secure User Authentication: Users can securely sign up, log in, and manage their accounts to access their personal assessment data.
- Assessment Form: A straightforward form allowing users to submit self-assessment scores (1-10) across Emotional, Cognitive, Motivation, and Relational domains for Day 0 and Day 14.
- Personalized Burnout Insights Tool: Utilizes AI to provide personalized feedback and insights based on the user's assessment scores, highlighting areas of potential concern or progress.
- Interactive Dashboard: Visual dashboard displaying current and historical assessment scores across all four domains, enabling easy comparison between Day 0 and Day 14 results.
- Secure Data Storage: All user assessment data is securely stored and managed using Firestore for reliable access and privacy.

## Style Guidelines:

- The primary color, #6688B3, is a soft, reassuring blue, conveying professionalism and calm, setting a composed tone for focused self-reflection. It is chosen for its ability to inspire trust and tranquility. Its HSL value is approximately H:220, S:40%, L:50%.
- The background color, #EFF4F8, is a very light desaturated blue derived from the primary hue, providing a clean and minimal canvas that enhances readability without distraction. Its HSL value is approximately H:220, S:15%, L:95%.
- An accent color, #3D8EA3, is a more vibrant yet still subdued blue-green. It creates a subtle contrast to highlight interactive elements and important information. Its HSL value is approximately H:190, S:60%, L:40%.
- The application will use 'Inter' for both headlines and body text. This sans-serif font offers a modern, objective, and highly legible appearance, contributing to the app's clean and minimal aesthetic ideal for professional users.
- Clean, minimalist line icons should be used to represent actions and data points, ensuring visual clarity and maintaining the professional aesthetic.
- A clean, spacious layout with a focus on clear hierarchy and content organization to ensure assessments and data visualizations are easily digestible.
- Subtle, smooth transitions for form submissions and dashboard updates to provide a responsive and pleasant user experience without being distracting.
